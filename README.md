# Firefly Coffee

Next.js website for Firefly Coffee — single-origin Kenyan specialty coffee.

## Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content

All content lives in `data/mock.ts` and is typed via `lib/types.ts`. To connect a CMS, replace the relevant fetchers in `lib/content.ts` — no component changes needed.
